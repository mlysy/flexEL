Introduction

flexEL is a computationally efficient R package called flexEL with source code in C++, which is flexible enough for users to solve any type of regression problems with minimum programming effort.

Code Structure

flexEL is currently capable to work with regression problems given fully observed covariates and fully observed or right-censored time-to-event resposes. The corresponding computations are handled separately by template classes el::InnerEL and el::InnerELC respectively. These two classes are designed as base classes that will inherit from a regression model class. Two examples of regression problems, mean regression and quantile regression are provided in this package, defined by the classes MeanRegModel and QuantRegModel respectively.
