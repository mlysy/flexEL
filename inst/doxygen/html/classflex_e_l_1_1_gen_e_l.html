Solve the dual problem via Newton-Raphson algorithm.

Parameters
[out]lambdaVector of length n_eqs containing the candidate solution to the dual optimization problem.
[in]GMoment matrix of size n_eqs x n_obs or n_eqs x (n_obs + supp_adj). If supp_adj = false, the former is required. If supp_adj = true and the former is provided, support adjustment is performed. If supp_adj = true and G.cols() == n_obs + 1, assumes that support has already been corrected.
[in]norm_weightsVector of weights of length n_obs or n_obs + 1 following same logic as above. Normalized to sum to one.
Note
Due to the difficulty of creating Eigen "pointers", current hack is to explicitly tell lambda_nr_impl whether to use external or internal G and norm_weights.
